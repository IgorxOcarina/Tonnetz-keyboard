# Tonnetz-keyboard
a simple puredata patch to be opened in plugdata as a vst to use your keyboard as a "tonnetz midi instrument"

To run this first install plugdata as a VST from the site https://plugdata.org/download.html
After that download the tonnetz_keyboard.pd file from this repository.
In your prefered DAW run plugdata as a VST and midiroute the output of it to your VST that generates audio
